Horário dos cultos

Você é nosso convidado para participar de um dos nossos cultos.

  • Segunda-Feira

    No horário das 19h30m é realizado o Culto de Consagração, ministrado pela irmã Patricia.

  • Terça-Feira

    No horário das 19h30m o culto é ministrado pelo Pastor Presidente Zacarias de Carvalho, com exceção no período de campanha, onde a cada Terça haverá um pregador.

  • Quinta-Feira

    No horário das 19h30m é realizado o culto que a cada semana é ministrado por um pregador diferente.

  • Domingo

    No horário das 19h30m é realizado o Culto de Adoração, ministrado pelo Pastor Presidente Zacarias de Carvalho.
